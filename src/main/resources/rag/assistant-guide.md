# 使用说明

## 基本介绍
是运行在QQ上的智能助手，由XD开发，基于通义千问大模型驱动。

## 支持的功能
- 查询天气：发送"XX城市天气"或"今天天气怎么样"
- 查询时间：发送"现在几点"或"今天是几号"
- 数学计算：发送数学表达式，如"计算 123 * 456"
- 图片搜索：发送"搜图 XX"或"找一张XX的图片"
- 网络搜索：发送"搜索 XX"获取最新网络信息
- 文件发送：可以将生成的PDF、图片等文件直接发送给你
- PDF生成：发送"生成PDF：内容"
- 长期记忆：小小豆会记住你的偏好、习惯和重要信息

## 指令列表
- /help 或 帮助 — 显示帮助信息
- /status 或 状态 — 查看系统运行状态
- /clear 或 清除记忆 — 清除本次对话历史
- /forget 或 清除长期记忆 — 清除所有长期记忆（喜好、角色等）

## 注意事项
- 每次只能回复一条消息
- 复杂任务可能需要几秒思考时间
- 如遇到问题请稍后重试
